# PotPlayer yt-dlp Chzzk Installer v1.0.0

## 한국어

치지직, YouTube, 네이버 등 yt-dlp 지원 사이트 URL을 PotPlayer에서 열기 위한 원클릭 설치기 첫 릴리스입니다.

포함 내용:

- `install-potplayer-ytdlp.cmd`
- `update-potplayer-ytdlp.cmd`
- `check-potplayer-ytdlp.cmd`
- 공식 PotPlayer yt-dlp 확장 다운로드
- 공식 yt-dlp 최신 릴리스 다운로드
- 기존 파일 자동 백업
- `chzzk:live`, `chzzk:video` 지원 확인

주의:

- PotPlayer를 끈 뒤 실행하세요.
- 관리자 권한이 필요합니다.
- Twitch 720p/source-quality 우회용 도구가 아닙니다.

## English

Initial release of a one-click installer for PotPlayer's yt-dlp media parser extension.

Included:

- Installer, updater, and checker CMD files
- Official PotPlayer yt-dlp extension download
- Official latest yt-dlp.exe download
- Automatic backup of existing files
- Chzzk live/video extractor verification

Notes:

- Close PotPlayer before running.
- Administrator permission is required.
- This is not a Twitch 720p/source-quality bypass.
